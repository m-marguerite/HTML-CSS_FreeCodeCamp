# 🎨 HTML-CSS_FreeCodeCamp

Une collection de mini projets HTML & CSS réalisés en autonomie pour pratiquer la structure sémantique, la mise en forme et les techniques de mise en page modernes du web.

## 🚀 Fonctionnalités

- Pages web statiques autonomes, chacune illustrant un concept HTML ou CSS spécifique
- Mise en page avec **Flexbox** et **CSS Grid** pour des layouts modernes et responsives
- Animations et effets visuels réalisés entièrement en CSS pur
- Utilisation des éléments HTML natifs : formulaires, tableaux, médias audio et vidéo
- Aucune dépendance externe — chaque projet s'ouvre directement dans le navigateur

## ⚙️ Prérequis

- Un navigateur web moderne (Chrome, Firefox, Edge, Safari)
- Aucun serveur requis — les projets s'ouvrent directement via le fichier `index.html`


## 🛠️ Technologies utilisées

- **HTML5** — Structure sémantique des pages, formulaires, tableaux et médias natifs
- **CSS3** — Mise en forme, typographie, couleurs, animations et transitions visuelles
- **Flexbox** — Alignement et distribution des éléments en une dimension
- **CSS Grid** — Mise en page en deux dimensions pour des layouts complexes

## 🎯 Objectif pédagogique

Ce projet illustre les fondamentaux du développement front-end en HTML et CSS pur :

- **Structure sémantique** : utilisation des balises HTML5 adaptées au contenu (`header`, `nav`, `main`, `section`, `article`, `footer`)
- **Mise en page moderne** : pratique de Flexbox et CSS Grid sur des cas concrets variés
- **Animations CSS** : création d'effets visuels et de transitions sans JavaScript
- **Formulaires HTML** : conception de formulaires accessibles avec différents types de champs
- **Responsive design** : adaptation des pages à différentes tailles d'écran
